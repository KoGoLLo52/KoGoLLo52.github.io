YTDLP-UI (WIP)

This may be the simplest one, as it is just a user interface (UI). The UI is designed to assist users with yt-dlp, as I found the video downloader tool to be difficult to use with console commands. Using commands in the console can be uncomfortable for some, so this UI allows users to easily utilize the tool.
